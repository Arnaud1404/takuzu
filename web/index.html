Takuzu

Game Rules

We have a grid that needs to be filled with gray and brown cats while follwing these rules:

  • There cannot be 3 or more cats of the same color in a given row or column.
  • There must not be more than half of the cats in a row or column of the same color.

We can add the following options to make the game harder:

  • Unique: There cannot be two identical rows or columns.
  • Wrapping: The game wraps around, meaning the cells at both ends are consecutive.

Controls

Cats with a crown are immutable, meaning they cannot be modified
Placing a gray cat:

  • PC: Left Click
  • Mobile: Short Click
Placing a brown cat :
  • PC: Right click
  • Mobile: Long Click
Clicling again on a square will empty it.
