Takuzu

Group project spanning the whole college year 2022-2023

Takuzu, also known as Binairo, is a logic puzzle involving placement of two symbols on a rectangular grid. The objective is to fill the grid with 1s and 0s, where there is an equal number of 1s and 0s in each row and column and no more than two of either number adjacent to each other.

Game Rules

We have a grid that needs to be filled with gray and brown cats while following these rules:

  • There cannot be 3 or more cats of the same color in each row and column.
  • There must be an equal number of cats in each row and column.

We can add the following options to make the game harder:

  • Unique: There cannot be two identical rows or columns.
  • Wrapping: The game wraps around, meaning the cells at both ends are consecutive.

Controls

Cats with a crown are immutable, meaning they cannot be modified
Placing a gray cat:

  • PC: Left Click
  • Mobile: Short Click
Placing a brown cat :
  • PC: Right click
  • Mobile: Long Click
Clicling again on a square will empty it.
